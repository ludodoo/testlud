Overview

MuK Documents is a module to create, manage and view files directly within Odoo. MuK Documents Access enables access control groups. These groups can be used to customize the security of your document management system.

Access Groups

Groups can be set to any directory or file. The users of a group are defined by a job or a given Odoo group.

Group Inheritance

Groups are inherited by any subdirectory or file. In case this is not the wanted behavior the mechanisms can be turned off by unchecking the checkbox.
